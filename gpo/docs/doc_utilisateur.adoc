= Présentation générale, rôle de l'application

== Présentation

Cette *application* est développée par *Pixel Rift* et permet de gérer des *tournois Esport*. Elle est développée avec *l'idée* de pouvoir proposer un *système de gestion* de tournoi directement pour les *jeux des consoles* de l'entreprise, mais l'application est *flexible* et permet d'être utilisée par *n'importe* quel type d'Esport. 

== Rôle

Ainsi, *l'application* permet notamment de pouvoir *créer plusieurs tournois* à la fois, sur des *jeux différents* et avec des *équipes différentes* constituées de *joueurs uniques*. Elle propose également une *gestion simple* des tournois grâce à la possibilité de *créer un staff*, avec des admins etc...

==== Manière n°2 : en allant voir les équipes

Pour consulter une inscription d'une équipe à un tournoi depuis le menu Equipe, vous pouvez partir de n'importe qu'elle page.
*Attention* il est très important d'être connecté au préalable que ce soit en administrateur ou en organisateur.
Nous sommes ici connecté en administrateur.

image::image_rendus/screen_pour_doc_util/page_d_acceuil.png[600, 400]

Pour aller voir les équipes inscrites, il est est nécessaire de se rendre sur le menu des Equipe. Pour se faire cliquez sur l'onglet
"Tournois" dans la barre du menu, puis sur "gestion des équipes".

image::image_rendus/screen_pour_doc_util/selection_menu_equipe.png[600, 400]

Vous voici dans le menu des équipes.

image::image_rendus/screen_pour_doc_util/page_d_acceuil_equipe.png[600, 400]

Pour voir les tournois dans lequels une équipe est inscrite,  sélectionner l'équipe que vous souhaitez regarder. Sur votre droite apparait tous les tournois auquel l'équipe est inscrite. Dans notre exemple, on peut voir que l'équipe Artic Bears est inscrite à trois tournois actuellement (CS Masters, Pro League et Valorant Open).

=== Modifié une équipe d'un tournoi (Ewen Ferreira)

Pour modifier une équipe, il est est nécessaire de se rendre sur le menu des Tournois. Pour se faire cliquez sur l'onglet
"Tournois" dans la barre du menu, puis sur "gestion des tournois".

image::image_rendus/screen_pour_doc_util/section_menu_tournois.png[600, 400]

Vous voici dans le menu des tournois.

image::image_rendus/screen_pour_doc_util/page_d_acceuil_tournois.png[600, 400]

Pour modifier une équipe à un tournoi, sélectionner le tournoi dans lequel vous souhaitez modifier votre équipe et cliquer sur le bouton "Gérer les inscriptions".

image::image_rendus/screen_pour_doc_util/page_d_acceuil_tournois_selectionner.png[600,400]

Vous voici dans le menu des inscriptions pour le Tournois que vous avez sélectionner. Dans notre exemple, nous pouvons voir les équipes inscrites au tournois "CS Masters". +
A présent, cliquer sur l'équipe que vous voulez modifier l'équipe choisi ici sera Titan Esports, puis sur le bouton  _modifier les inscriptions_. 

image::image_rendus/screen_pour_doc_util/modifier_tournoi.png[600,400]

Vous voici dans le menu de modification, saisissez les modifications en respectant les conditions de créations et appuyer sur _Valider_ pour enregistrer vos modification ou _Annuler_ pour rien changer.

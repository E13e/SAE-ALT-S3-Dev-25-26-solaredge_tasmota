# Sprint 1

## Fonctionnalités implémentées

Il est possible de lancer le projet docker en lançant la commande "./launch.sh" qui se trouve dans le répertoire docker (présent dans le zip). 
Pour des information plus précises sur le lancement voir la documentation utilisateur : https://github.com/E13e/SAE-ALT-S3-Dev-25-26-solaredge_tasmota/blob/main/gpo/docs/doc_utilisateur.adoc .  
Une fois lancée les données des panneaux solaires sont accessibles via ce lien http://127.0.0.1:8080/php/.
